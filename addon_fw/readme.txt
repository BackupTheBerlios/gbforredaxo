Das Addon-Framework wird für das Gästebuch benötigt.
Diese Version soll nur sicherstellen eine funktionierende Version für das Gästebuch zu haben.
Durch Änderungen an der Originalversion aus dem Redaxo-CVS könnte es sonst zu Problemen kommen.

Es wird empfohlen zuerst die Redaxo-CVS-Version des Addon-Framework zu verwenden.
Sollte es damit Probleme geben und das Addon-Framework sonst nirgends benötigt werden, kann diese Version hier verwendet werden.
